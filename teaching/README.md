## Workshop Formats

### 1-Hour Introduction (`workshops/1-hour-intro/`)
**Audience:** General public, policy enthusiasts  
**Prerequisites:** None  
**Covers:** What is fiscal simulation, quick demo, Q&A  
**Materials:** [WORKSHOP_GUIDE.md](workshops/1-hour-intro/WORKSHOP_GUIDE.md)

### Half-Day Policy Workshop (`workshops/half-day-policy/`)
**Audience:** Policy students, government staff  
**Prerequisites:** Basic economics knowledge  
**Covers:** Budget basics, healthcare analysis, hands-on exercises  
**Materials:** [WORKSHOP_GUIDE.md](workshops/half-day-policy/WORKSHOP_GUIDE.md)

### Full-Day Technical Workshop (`workshops/full-day-technical/`)
**Audience:** Researchers, data scientists  
**Prerequisites:** Python, statistics  
**Covers:** Monte Carlo methods, API integration, custom policies  
**Materials:** [WORKSHOP_GUIDE.md](workshops/full-day-technical/WORKSHOP_GUIDE.md)

## Course Integration

PoliSim can be integrated into:

1. **Public Policy courses** - Real-world policy simulation
2. **Economics courses** - Fiscal policy modeling
3. **Data Science courses** - Monte Carlo case study
4. **Political Science courses** - Budget process education

See [INSTRUCTOR_GUIDE.md](INSTRUCTOR_GUIDE.md) for detailed syllabi.

## Teaching Mode

Enable teaching mode in the dashboard for:
- Enhanced tooltips
- Guided tours
- Simplified interface options

```python
# Enable in config.yaml
ui:
  teaching_mode: true
  difficulty_level: beginner  # beginner, intermediate, advanced
```
