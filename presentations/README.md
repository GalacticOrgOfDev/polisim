# PoliSim Presentations

Conference and outreach presentation materials.

## Contents

| File | Description | Use Case |
|------|-------------|----------|
| `polisim_main_deck.pptx` | 30-slide comprehensive presentation | Conferences, seminars |
| `polisim_poster.pdf` | Academic poster (48"x36") | Poster sessions |
| `demo_script.md` | Live demo walkthrough | Presentations |
| `one_pager.pdf` | Single-page fact sheet | Handouts, emails |

## Main Presentation Deck

### Sections
1. **Introduction** (slides 1-4): Problem, solution, overview
2. **Architecture** (slides 5-10): Technical foundation
3. **Demos** (slides 11-20): Healthcare, Social Security, comparisons
4. **Validation** (slides 21-24): CBO/SSA reconciliation
5. **Community** (slides 25-28): Open source, contributions
6. **Conclusion** (slides 29-30): Roadmap, call to action

### Customization
- Editable in PowerPoint or Google Slides
- All charts exportable as SVG/PNG
- Speaker notes included
