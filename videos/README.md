## Recording Guidelines

### Equipment Recommendations
- Screen recording: OBS Studio, Camtasia, or similar
- Microphone: USB condenser mic recommended
- Resolution: 1920x1080 minimum

### Recording Checklist
- [ ] Clean desktop (hide icons)
- [ ] Dashboard running with sample data loaded
- [ ] Terminal cleared
- [ ] Notifications disabled
- [ ] Script reviewed

### Branding
- Use PoliSim color scheme
- Include logo in lower third
- Consistent font: Inter or system sans-serif

## Assets Directory

`assets/` contains:
- Logo files (SVG, PNG)
- Intro/outro templates
- Lower third graphics
- Background music (royalty-free)
