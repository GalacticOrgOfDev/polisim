# PoliSim Educational Notebooks

Interactive Jupyter notebooks for learning fiscal policy simulation with PoliSim.

## Curriculum Overview

| # | Notebook | Title | Duration | Audience | Status |
|---|----------|-------|----------|----------|--------|
| 01 | `01_welcome_to_polisim.ipynb` | Welcome to PoliSim | 15-20 min | Beginners | ✅ Complete |
| 02 | `02_federal_budget_basics.ipynb` | Federal Budget Basics | 30-40 min | Students, Citizens | ✅ Complete |
| 03 | `03_healthcare_policy_analysis.ipynb` | Healthcare Policy Analysis | 45-60 min | Policy Students | ✅ Complete |
| 04 | `04_social_security_deep_dive.ipynb` | Social Security Deep Dive | 45-60 min | Policy Analysts | ✅ Complete |
| 05 | `05_monte_carlo_uncertainty.ipynb` | Monte Carlo & Uncertainty | 60-90 min | Quant Researchers | ✅ Complete |
| 06 | `06_tax_policy_modeling.ipynb` | Tax Policy Modeling | 45-60 min | Economics Students | ✅ Complete |
| 07 | `07_policy_extraction.ipynb` | Policy Extraction | 30-45 min | Policy Analysts | ✅ Complete |
| 08 | `08_api_integration.ipynb` | API Integration | 30-45 min | Developers | ✅ Complete |
| 09 | `09_custom_policy_design.ipynb` | Custom Policy Design | 60-90 min | Advanced Users | ✅ Complete |
| 10 | `10_capstone_analysis.ipynb` | Capstone Analysis | 2-3 hours | Course Projects | ✅ Complete |

**Total Curriculum Time**: ~8-12 hours (complete all notebooks)

## Getting Started

```bash
# Install Jupyter
pip install jupyter

# Launch notebooks
jupyter notebook notebooks/
```

## Prerequisites

- Python 3.9+
- PoliSim installed (`pip install -r requirements.txt`)
- Basic Python knowledge (for notebooks 1-4)
- Statistics knowledge (for notebook 5)
- Programming experience (for notebooks 8-9)

## Learning Paths

### Path 1: Policy Analyst (3-4 hours)
Notebooks: 01 → 02 → 03 → 04 → 06

### Path 2: Quantitative Researcher (4-5 hours)
Notebooks: 01 → 02 → 05 → 06 → 09

### Path 3: Developer (2-3 hours)
Notebooks: 01 → 08 → 09

### Path 4: Full Course (8-10 hours)
All notebooks in order
